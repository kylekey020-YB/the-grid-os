## Purpose

The Daily Command Brief is the first page Mission Commander sees every morning.

It reduces coordination by forcing every division to report through the same schema and ending with one recommendation.

Every section reports:

- Status
- Signal
- Evidence
- Blocker
- Next Action
- Related Mission ID
- Obsidian Note
