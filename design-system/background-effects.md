# Background Effects

Current Version: v0.3.0

Background effects establish THE GRID's command-center identity. They should remain restrained behind the content.

## Approved Effects

| Effect | Use |
| --- | --- |
| Fine grid overlay | Mission Control hero and command surfaces |
| Single gradient border line | Top edge of major command surfaces |
| Low-opacity accent wash | Large section background depth |
| Subtle scan line | Future data surfaces, if it does not reduce readability |

## Rules

- Background effects must not reduce text contrast.
- Do not add decorative orbs, bokeh blobs, or unrelated abstract shapes.
- Do not make every section a hero surface.
- Keep page sections full-width or unframed unless the content is a repeated card, modal, or tool surface.
